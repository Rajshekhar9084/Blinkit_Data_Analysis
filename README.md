# 🛒 Blinkit Sales Dashboard (Excel)

An interactive Excel dashboard built to analyze Blinkit sales performance using Pivot Tables, Pivot Charts, and Slicers.

## 📊 Key Metrics

- Total Sales: ₹295,185
- Total Quantity Sold: 1,773
- Total Profit: ₹20,785
- Total Customers: 200
- Profit Margin: 7.04%
